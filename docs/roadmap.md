# Roadmap

## MVP
- Local vault encrypted in the browser.
- Master password unlock flow.
- Add, edit, delete, and search credentials.
- Password generator with configurable length and character sets.
- Auto-lock after inactivity.
- Encrypted export and import.
- Basic settings screen for security-related preferences.
- Clear README-ready technical narrative for GitHub.

## Version 1.1
- Stronger vault metadata model with tags and favorites.
- Copy-to-clipboard UX with auto-clear timer.
- Password strength feedback and reuse warnings.
- Better empty states and onboarding.
- Session timeout configuration.
- Keyboard shortcuts for power users.

## Version 1.2
- Entry history or audit trail for local changes.
- Secure notes and custom fields.
- Per-item attachments, if the storage model supports it safely.
- Improved search and filtering.
- Optional biometric-style unlock where the browser and platform support it.

## Version 2.0
- Migration framework for encrypted vault schema changes.
- More polished export/import flows with validation and error recovery.
- Offline-first improvements for larger vaults.
- Accessibility pass focused on keyboard navigation and screen readers.
- Stronger test coverage for crypto, persistence, and lock behavior.

## Portfolio Value
- Demonstrates full-stack product thinking even if the first version is local-only.
- Shows applied security knowledge through key derivation, encryption, and secure storage.
- Gives you something concrete to discuss in interviews: threat model, tradeoffs, and limitations.
- Supports a strong GitHub story because the project can be documented in a way that reads like a small security-focused product.

## GitHub Story
- The repo should make it obvious that this is a carefully scoped educational project.
- Each major milestone should be tied to a visible feature and a technical decision.
- The changelog should emphasize security improvements, usability improvements, and architectural decisions.
- Issues can be used to show roadmap thinking and iterative delivery.
